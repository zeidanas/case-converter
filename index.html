Instructions

To use the case converter, follow these simple steps:


  • Enter a name for the file.
  • Enter your text into the large text box.
  • Click the 'Upper Case' button to capitalize every letter in the text.
  • Click the 'Lower Case' button to lower case every letter in the text.
  • Click the 'Proper Case' button to capitalize the first letter of each word in the text.
  • Click the 'Sentence Case' button to capitalize the first letter of the first word of each sentence in the text.
  • Click the 'Save Text File' button to save the text in a text file.
